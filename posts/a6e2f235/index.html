docker容器固定IP地址

2025-08-26
473- 1m- - -
教程 > 群晖-群晖

前提条件

  • 已安装 Docker 环境
  • 知道目标容器名称或 ID

操作步骤

  1. 创建自定义网络(如果已有请忽略)

  2. 停止目标容器

  3. 重新启动容器并指定固定 IP

  4. 验证 IP 是否固定

注意事项

  • 固定 IP 必须在自定义网络的网段范围内

  • 同一网络内的 IP 不能重复

  • 重启 Docker 服务或容器，IP 保持不变

  • 若需修改 IP，需删除容器后重新创建并指定新 IP

  • 方法：查看指定网络中的所有 IP 分配

    1. 先列出所有网络，确认目标网络名称
